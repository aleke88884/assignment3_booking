# SmartBooking - Система Бронирования

Production-ready система бронирования для малого и среднего бизнеса (бани, бассейны, спортивные площадки, конференц-залы)

## Возможности

- Управление ресурсами с категориями и фотографиями
- Система бронирования с проверкой доступности
- Загрузка и хранение фотографий в S3/MinIO  
- Гибкая ценовая политика (разные тарифы)
- Расписание работы ресурсов
- Отзывы и рейтинги
- Аутентификация пользователей
- RESTful API с Swagger документацией

## Быстрый старт

```bash
# Запустить все сервисы
make up

# Посмотреть логи
make logs

# Остановить
make down
```

Откройте:
- API: http://localhost:8080
- Swagger: http://localhost:8080/swagger/
- MinIO: http://localhost:9001 (minioadmin/minioadmin)
